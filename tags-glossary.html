Glossário HTML (Tags)

Tags mais utilizadas

class=*
Atribuí uma classe ao elemento
style=*
Atribuí elementos CSS dentro da Tag
lang=*
Define o idioma principal do elemento
hidden
Oculta o elemento
align=”…”
Permite definir o padrão de alinhamento desse elemento, como por (ex: right, center, left e justify)
width=*
Define uma largura para o elemento;
height=*
Define uma altura para o elemento.
